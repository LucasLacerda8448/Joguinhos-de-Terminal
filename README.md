# Por que eu fiz isso?
Esse repositório foi criado como uma forma de treinar um pouco minhas habilidades em Python e também para explorar um pouco mais como eu poderia criar joguinhos simples que pudessem rodar sem necessitar de uma interface gráfica, muito disso devido ao meu interesse com a área de jogos.

# Se não tem interface gráfica, como eu faço os visuais?
Não nada muito Uau, porém todos os códigos usam caracteres ASCII e Alt Code como os visuais para os jogos.
